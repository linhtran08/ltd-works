# [LTD Works](https://ltd-portfolio.vercel.app)

The app for works, schedule, and more by LTD

## Installation

Use the package manager [pnpm](https://pnpm.io/) to install LTD Works.

```bash
npm install -g pnpm
```
```bash
pnpm install 
```

## Start

To start the LTD Works app 
```bash
pnpm start
```
